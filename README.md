# ForkMaster - Ethical Project Forking Platform

ForkMaster helps builders identify opportunities to ethically fork existing projects and build differentiated MiniApps on Base.

## Features

- **Niche Opportunity Finder**: Analyzes popular projects and identifies underserved niches
- **Ethical Forking Framework**: Step-by-step guide for legal and ethical compliance
- **Value-Add Feature Generator**: AI-powered suggestions for feature enhancements
- **Rapid Differentiation Tester**: Farcaster frames for community validation

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   ```bash
   cp .env.example .env.local
   ```
   
   Fill in your API keys:
   - `NEXT_PUBLIC_MINIKIT_API_KEY`: Your MiniKit API key
   - `OPENROUTER_API_KEY`: Your OpenRouter API key for AI analysis

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## How It Works

1. **Enter a project URL** you want to analyze for forking opportunities
2. **Get AI-powered analysis** including niche identification and ethical considerations
3. **Review suggested features** that would add value beyond the original
4. **Generate test frames** to validate your ideas with the Farcaster community
5. **Build with confidence** knowing you're following ethical practices

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **MiniKit** for Base integration
- **OnchainKit** for wallet functionality
- **OpenAI/OpenRouter** for AI analysis
